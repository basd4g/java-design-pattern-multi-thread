# Java言語で学ぶデザインパターン入門 マルチスレッド編

## 環境構築

```sh
$ cat /etc/issue
Ubuntu 20.04 LTS \n \l

$ sudo apt update
$ sudo apt install openjdk-11-jdk
$ which java
/usr/bin/java
```

## 実行

```sh
git clone git@github.com:basd4g/java-design-pattern-multi-thread.git
cd java-design-pattern-multi-thread/hello
javac Hello.java
java Hello
```
